* Serve
bundle exec jekyll serve
  --drafts        (add drafts)
  --livereload    (immediately reload upon changes in .md files)
  --host=0.0.0.0  (listen on all incoming connections)

* Invisible posts
To hide a published post, set ~hidden: true~ in the header data.

* Updating theme defaults
Find the Ruby gem location using ~bundle info --path minima~. All the defaults are stored there. Copy the ones you want to change over to the project folder.
